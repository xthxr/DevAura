## ‚ö†Ô∏è Special Note: Password with Special Characters

If your Supabase password has special characters (like `@`, `#`, `!`, etc.), you need to URL-encode them:

| Character | Encoded |
|-----------|---------|
| `@` | `%40` |
| `#` | `%23` |
| `!` | `%21` |
| `$` | `%24` |
| `%` | `%25` |
| `^` | `%5E` |
| `&` | `%26` |
| `*` | `%2A` |

Example:
- Password: `MyPass@123!`
- Encoded: `MyPass%40123%21`
- Full URL: `postgresql://postgres:MyPass%40123%21@db.xxx.supabase.co:5432/postgres`
